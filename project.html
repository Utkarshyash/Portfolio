Alien Invasion

About:

Alien Invasion is a space arcade game, where the goal is to prevent earth being invaded by alien Unidentified Flying Objects (aliens). The player controls a moving base and has some missiles to shoot down as many aliens as possible in order to win the game. In a nutshell it is a classic type arcade game based on destroying aliens in a far away universe. Game is written using pygame python package.The user spaceship has multi-colored bullets to shoot the aliens.Aliens too have bullets to defeat the user's spaceship,which make this game competitive and interesting.
